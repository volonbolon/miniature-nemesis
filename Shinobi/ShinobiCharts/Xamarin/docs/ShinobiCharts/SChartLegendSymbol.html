SChartLegendSymbol Class

Abstract base class for symbols representing the series for display on the Legend.

Syntax

[MonoTouch.Foundation.Register("SChartLegendSymbol", true)]
public abstract class SChartLegendSymbol : MonoTouch.UIKit.UIView

Remarks

The legend symbols are the visual representations of a series in the chart legend. SChartLegendSymbol is the base class for these symbols.

As a part of adopting the SChartLegendItem protocol, a series may optionally return a symbol object to represent itself. If you are using one of the series types provided by ShinobiCharts, you shouldn't need to create legend symbols yourself. All of the series types return an appropriate symbol which will represent them in the chart legend. These predefined symbols are:

  • ShinobiCharts.SChartLegendSymbolLineSeries - represents a line series.
  • ShinobiCharts.SChartLegendSymbolColumnSeries - represents a column series.
  • ShinobiCharts.SChartLegendSymbolBarSeries - represents a bar series.
  • ShinobiCharts.SChartLegendSymbolScatterSeries - represents a scatter series.
  • ShinobiCharts.SChartLegendSymbolPieSeries - represents a pie series.
  • ShinobiCharts.SChartLegendSymbolDonutSeries - represents a donut series.
  • ShinobiCharts.SChartLegendSymbolBandSeries - represents a band series.
  • ShinobiCharts.SChartLegendSymbolCandlestickSeries - represents a candlestick series.
  • ShinobiCharts.SChartLegendSymbolOHLCSeries - represents a OHLC series.
