SChartMultiXDataPoint Class

Represents cartesian data points which have multiple x values.

Syntax

[MonoTouch.Foundation.Register("SChartMultiXDataPoint", true)]
public class SChartMultiXDataPoint : SChartDataPoint

Remarks

Examples of series which could have multiple x values for each data point are:
  • ShinobiCharts.SChartBandSeries
  • ShinobiCharts.SChartCandlestickSeries
  • ShinobiCharts.SChartOHLCSeries
