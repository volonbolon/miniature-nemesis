SChartMultiYDataPoint Class

Represents cartesian data points which have multiple y values.

Syntax

[MonoTouch.Foundation.Register("SChartMultiYDataPoint", true)]
public class SChartMultiYDataPoint : SChartDataPoint

Remarks

Examples of series which could have multiple y values for each data point are:
  • ShinobiCharts.SChartBandSeries
  • ShinobiCharts.SChartCandlestickSeries
  • ShinobiCharts.SChartOHLCSeries
