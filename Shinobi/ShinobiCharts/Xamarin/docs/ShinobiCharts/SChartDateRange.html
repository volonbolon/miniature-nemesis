SChartDateRange Class

An instance of SChartRange that is specific to NSDate - with each element in the range represented by an NSDate object.

Syntax

[MonoTouch.Foundation.Register("SChartDateRange", true)]
public class SChartDateRange : SChartRange

Remarks

This is the likely range of choice for an SChartDateTimeAxis.
