ShowSymbols Property

Defines whether symbols will be shown in the legend.

Syntax

public bool ShowSymbols { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

The default value is true (symbols will be shown). Symbols are collected from each of the chart series using the SChartLegendItem protocol. If this is set to false, each chart series will be represented in the legend by a text label only.
