SChartNumberRange Class

Represents numerical ranges on a chart axis (SChartAxis).

Syntax

[MonoTouch.Foundation.Register("SChartNumberRange", true)]
public class SChartNumberRange : SChartRange

Remarks

This is the likely range of choice for a SChartNumberAxis, but numerical ranges can also be applied to other axis types, including SChartDateTimeAxis and SChartCategoryAxis.
