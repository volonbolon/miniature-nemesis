SChartRange Class

Represents a range of values on a chart.

Syntax

[MonoTouch.Foundation.Register("SChartRange", true)]
public abstract class SChartRange : MonoTouch.Foundation.NSObject

Remarks

It is used by the chart axes, which are defined in SChartAxis. SChartRange is the base class for the concrete range types which are available in ShinobiCharts. The concrete types are:
  • SChartNumberRange - represents numerical ranges.
  • SChartDateRange - represents date/time ranges.
