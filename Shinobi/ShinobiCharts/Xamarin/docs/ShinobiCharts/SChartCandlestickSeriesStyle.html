SChartCandlestickSeriesStyle Class

Manages the appearance of an instance of SChartCandlestickSeries on the chart.

Syntax

[MonoTouch.Foundation.Register("SChartCandlestickSeriesStyle", true)]
public class SChartCandlestickSeriesStyle : SChartOHLCSeriesStyle

Remarks

Each data point in a candlestick series is visualised as a thick candle between the open and close values, with 'wicks' indicating the high and low positions.

SChartCandlestickSeriesStyle inherits some of its behavior from its parent, ShinobiCharts.SChartOHLCSeriesStyle. This controls things like the colors which are used for data points when their data values are rising or falling. Have a look at the API documentation of the parent class for more information.

This style object allows you to configure properties which are specific to the appearance of candlestick charts. These are:

  • The color and width of the sticks (or 'wicks') at either end of the candle. See SChartCandlestickSeriesStyle.StickColor or SChartCandlestickSeriesStyle.StickWidth for more information.
  • The color and width of the outline of the candle. See SChartCandlestickSeriesStyle.OutlineColor and SChartCandlestickSeriesStyle.OutlineWidth for more information.
