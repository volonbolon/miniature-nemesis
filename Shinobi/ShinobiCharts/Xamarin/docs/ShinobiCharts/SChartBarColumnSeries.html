SChartBarColumnSeries Class

Abstract base class for column and bar series.

Syntax

[MonoTouch.Foundation.Register("SChartBarColumnSeries", true)]
public abstract class SChartBarColumnSeries : SChartCartesianSeries

Remarks

To display a bar or column series on the chart init and use one of the classes Shinobicharts.SChartBarSeries or Shinobicharts.SChartColumnSeries respectively.
