SChartBoxGestureStyle Class

Visual style properties for the box zoom feature.

Syntax

[MonoTouch.Foundation.Register("SChartBoxGestureStyle", true)]
public class SChartBoxGestureStyle : MonoTouch.Foundation.NSObject, NSCopying

Remarks

If you set the ShinobiChart.GesturePanType property on a chart to SChartGesturePanType.BoxDraw, the chart will draw a box on the chart during a pan. When the pan is finished, the chart will zoom to the limits of the box.

The SChartBoxGestureStyle class manages the look and feel of the box gesture. An instance of the class is contained by the chart's theme. To update the look of the box gesture on a chart, you could use the following code:
Example
chart.theme.boxGestureStyle.boxLineColor = [UIColor redColor];
[chart applyTheme];

The definition of the chart theme can be found in ShinobiCharts.SChartTheme.
