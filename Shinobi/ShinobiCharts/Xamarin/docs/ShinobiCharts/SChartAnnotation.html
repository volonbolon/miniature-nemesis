SChartAnnotation Class

A UIView that can be displayed on a chart - maintaining aspect ratio at all times and position in accordance with the panning of the data range.

Syntax

[MonoTouch.Foundation.Register("SChartAnnotation", true)]
public class SChartAnnotation : MonoTouch.UIKit.UIView

Remarks

There are several predefined annotations for convenience - these can be created using the methods below and are bands, lines and text.

An SChartAnnotation is fixed to a single point on the chart which means that it will pan but will remain a fixed size regardless of zoom level. If you would like to annotate an area that will scale as the data range is zoomed use ShinobiCharts.SChartAnnotationZooming, which is anchored between two limits on either axis.

An annotation can be attached to one of the x axes, and one of the y-axes in the chart. This is done using the SChartAnnotation.XAxis and SChartAnnotation.YAxis properties on the annotation. If an annotation is attached to an axis, the position of the annotation on the chart will correspond to its value on that axis. The position of the annotation on the x axis will correspond to SChartAnnotation.XValue and its position on the y axis will correspond to SChartAnnotation.YValue.

Position Property

Determines if the annotation should be drawn in front of or underneath your chart data.

Syntax

public SChartAnnotationPosition Position { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

This controls which visual layer the annotation is drawn in. In most cases, we will want the annotation to be drawn in front of the chart data, so this defaults to SChartAnnotationPosition.AboveData, showing the annotation on top off the data.
