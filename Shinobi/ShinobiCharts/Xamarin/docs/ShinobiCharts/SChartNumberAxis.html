SChartNumberAxis Class

An axis that works with numeric data points

Syntax

[MonoTouch.Foundation.Register("SChartNumberAxis", true)]
public class SChartNumberAxis : SChartAxis

Remarks

When mapping coordinates it will cast each object to an NSNumber.

The frequency values for tick marks are expected to be NSNumber objects.
