SChartBarColumnSeriesStyle Class

SChartBarColumnSeriesStyle manages the look of bar and column series on a chart.

Syntax

[MonoTouch.Foundation.Register("SChartBarColumnSeriesStyle", true)]
public class SChartBarColumnSeriesStyle : SChartSeriesStyle

Remarks

Contains any properties which are common to both bar and column series.

For properties which are specific to bar series, you should use the ShinobiCharts.SChartBarSeriesStyle. For properties which are specific to column series, you should use the ShinobiCharts.SChartColumnSeriesStyle.

ShowAreaWithGradient Property

Set this property to true to add a gradient to the fill inside the bar/column.

Syntax

public bool ShowAreaWithGradient { get; set; }

Value

Documentation for this section has not yet been entered.

Remarks

If the area is not filled, this property won't have any effect. When a gradient is applied to the fill, the color starts at AreaColor, and finishes at AreaColorGradient. The gradient isn't configurable. It runs from left to right on column series, and from top to bottom on bar series. This defaults to true.
