SChartTimePeriod Class

Represents a period of time.

Syntax

[MonoTouch.Foundation.Register("SChartTimePeriod", true)]
public class SChartTimePeriod : MonoTouch.Foundation.NSObject

Remarks

It starts at a point in time, and has a length defined by an instance of SChartDateFrequency. Instances of SChartTimePeriod are used by the discontinuous date/time axis (defined in SChartDiscontinuousDateTimeAxis). The axis defines periods of time over which it will skip.
