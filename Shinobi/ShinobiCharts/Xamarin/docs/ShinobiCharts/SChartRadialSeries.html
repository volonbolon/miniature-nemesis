SChartRadialSeries Class

Abstract base class for series displaying data in a cirle, with each data point shown as a slice with an angle proportional to its X value.

Syntax

[MonoTouch.Foundation.Register("SChartRadialSeries", true)]
public abstract class SChartRadialSeries : SChartSeries

Remarks

SChartRadialSeries represents all radial series within a ShinobiChart. This is the base class for radial series, typically you would use one of its subclasses in your chart. Currently the following subclasses are available:

  • SChartDonutSeries - represents donut series.
  • SChartPieSeries - represents pie series.

Whereas datapoints for cartesian series require an x and a y value, radial series require a name and a value. The x value of a datapoint within a radial series is set as the name of the slice, and the y value is set as its magnitude. See ShinobiCharts.SChartRadialDataPoint for more information.
