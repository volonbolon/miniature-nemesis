This is the Xamarin.iOS bindings for ShinobiCharts.

The lib folder contains the ShinobiCharts.dll you will need to reference on your projects that intend to use ShinobiCharts.

The docs folder contains HTML documentation describing each class and it's members.
You can find a getting started guide for ShinobiCharts with Xamarin.iOS located in the Obj-C documentation (in the parent folder).

The samples folders contains some examples to help you get started, as well as some how-to samples illustrating how to use various features of the library.
